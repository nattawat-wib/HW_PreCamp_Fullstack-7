Global Attributes
Attribute Description
accesskey ใช้ระบุ shortcut เพื่อ fosuc หรือ active tag นั้นๆ
class ใช้เพื่อเพิ่ม class ของ css เข้ามาใน tag
contenteditable ตั้งค่าให้ user สามารถกำหนดค่าของ element นั้นๆได้ผ่านเว็บบราวเซอร์
data ใช้เพื่อเก็บค่าใดค่าหนึ่งไว้ใน tag สำหรับใช้งานต่อไป เช่น เขียน script
dir ใช้บอกลำดับของข้อมูล เนื้อหาใน webpage นั้น ๆ ที่มีการเรียกใช้
draggable ใช้กำหนด element ให้สามารถเคลื่อนย้ายได้
hidden ใช้กำหนดการส่งค่าตัวแปรผ่าน form รูปแบบหนึ่ง
id ใช้เพื่อกำหนดอัตลักษณ์หรือชื่อเฉพาะสำหรับอ้างอิงตัวตนของ Element ใดๆเพียง element เดียวในหน้าเว็บนั้น
lang ใช้เพื่อกำหนดภาษาให้กับ tag นั้นๆ
spellcheck กำหนดว่า element นั้นจะให้ทำการ spellcheck หรือไม่
style ใช้เพื่อเพิม css เข้ามาใน tag
tabindex ใช้กำหนดลำดับการ focus ของ element เมื่อเรากด tab
title ใช้กำหนดค่าข้อความที่จะแสดงเมื่อเรานำเมาส์ไปวางบน element นั้นๆ
translate ใช้กำหนดว่า จะแปลภาษให้อัตโนมัติหรือไม่
2.differentiation tag
Tag Similar Difference
<hr> & <br> ใช้เว้นพื้นที่ว่างสำหรับ element 2 สิ่ง hr : ใช้เส้นในการแบ่ง
br : เว้นพื้นที่ว่างไว้
<ul> & <ol> สำหรับทำ list รายการเพื่อแสดงในหน้าเว็บ ul : ใช้จุด (dot) ในการเริ่มต้น list
or : ใช้ตัวเลข (number) ในการเริ่มต้น list
<Select> &
<checkbox> &
<radio button> 		ใช้ทำรายการเพื่อให้ user เลือก select : เป็นเมนูแบบ dropdown ลงมาเลือกได้อย่างเดียว
checkbox : เป็นเมูนแบบ list เลือกได้มากกว่า 1 รายการ
ratio : เป็นเมนูแบบ list เลือกเได้พียง 1 รายการ
<div> & <span> สำหรับแยก element 2 สิ่งออกจากกัน div : ทำงานแบบ block level ตัดขึ้นบรรทัดใหม่ทันทีหากปิด tag
span : ทำงานแบบ inline level ยังคงอยู่ในบรรทัดเดิมแม้ปิด tag
<script> & <style> สำหรับเรียกใช้งานส่วนเสริมเพิ่มเติม scrpit : สำหรับเรียกใช้งานภาษา javascript
style : สำหรับเรียกใช้งาน CSS

file รับ input เป็นไฟล์ประเภทต่างๆจาก user
hidden ใช้สำหรับ developer ตั้งค่าข้อมูลที่ user ไม่สามารถดูและแก้ไขได้
image ใช้แสดงผลรูปภาพและทำหน้าที่เหมือน summit button
month รับ input เป็นข้อมูลประเภทเดือนจาก user

password รับ input เป็นรหัสผ่าน
radio สร้าง list ตัวเลือกเพื่อรับข้อมูลจากสิ่งที่ user เลือก เพียง 1 รายการ
range สร้างแถบเพื่อให้ user กำหนดค่าของ data ที่จะรับ input
reset สำหรับ user reset ค่าที่จะ input
search สำหรับ user input ข้อมูลที่ต้องการค้นหาจาก data base
submit ใช้เพื่อกดส่งข้อมูลไปประมวลผลต่อใน backend

time รับ input เป็นเวลา
url รับ input เป็น url
week รับ input เป็นข้อมูลประเภทสัปดาห์
